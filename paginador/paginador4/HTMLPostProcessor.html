HTMLPostProcessor

Description
Description | Vars (details) | Methods (details)

This class allow to insert in a database table data contained inside the global array $_POST and save files described by $_FILES.

It build an inserting/editing SQL query and perform file uploads.



ITALIAN:
La classe consente di inserire su una tabella di database i dati contenuti nell'array globale $_POST e salvare i file descritti da $_FILES. Costruisce una query SQL di inserimento o modifica ed effettua l'upload dei file.
